## 大学の授業の課題で作ったアラームアプリ(※齧った程度)
## アプリの名前  
 多機能アラーム  
## 機能  
1.アラームを止めるときに明るさが一定以上の時間明るくないとアラームが止まらない  
2.天気の情報によって音が変わる  
3.時間になるとライトが付く
